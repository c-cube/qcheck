Module QCheck.Print

The Print module offers combinators for printing generated values.

type 'a t = 'a -> string

val comap : ('a -> 'b) -> 'b t -> 'a t

comap f p maps p, a printer of type 'b, to a printer of type 'a by first converting a printed value using f : 'a -> 'b.
